# Deployment of Static Website on S3

## Description

This project is a simple AWS project to deploy static website on S3.

## Components

1. S3 Bucket
2. S3 Bucket Policy (Allowing public read access to the S3 bucket)
3. S3 Bucket Website Configuration (Setting up the website configuration for the S3 bucket)
4. S3 Bucket Object (Copying source files to S3 bucket)


Note: Tested with LocalStack.
